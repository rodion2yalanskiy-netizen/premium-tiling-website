## ПАРАМЕТРЫ ВЫЗОВА API

```python
# Вызов в боте (OpenRouter)
response = openrouter_client.chat.completions.create(
    model="anthropic/claude-3-haiku",          # быстро и дёшево
    # model="anthropic/claude-3.5-sonnet",     # если нужна максимальная точность
    messages=[
        {"role": "system", "content": PROMPT_REFINER_SYSTEM},
        {"role": "user", "content": f"Сырая задача от Родиона:\n\n{raw_task}"}
    ],
    max_tokens=1500,
    temperature=0.3    # низкая температура = точность > креативность
)
```

Стоимость: ~$0.0003 за рефайнинг (Haiku) | ~$0.003 за рефайнинг (Sonnet)
